	Location list on job posting for Job Board Manager.

== Description ==

"Job Board Manager - Locations" allowes you to display ajax list of predefined location on job edit page and available job list under any location for "Job Board Manager" plugin.


### Job Board Manager - Locations by http://pickplugins.com
* [Live Demo &raquo;](http://www.pickplugins.com/demo/job-board-manager/location/dhaka/)
* [Buy Premium &raquo;](http://www.pickplugins.com/item/job-board-manager-locations/)
* [Get Job Board Manager &raquo;](https://wordpress.org/plugins/job-board-manager/)

<strong>Plugin Features</strong>

* Single Location page.
* Location information display automatically form wikipedia.org
* Google map for location on header single location page.
* Available job list on single location page with pagination support.
* Ajax list of location when start typing on job submit form.

<strong>Premium Features</strong>

* Job count by location.
* Jobs in map view.
* Job expandable list by country and city.
* Location submit form.
* List of related location under same country on single location page.


<strong>Translation</strong>

Plugin is translation ready, you can contribute in translation. please contact with us http://pickplugins.com/contact/

== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Job Board Manager</strong>" activate it.<br />
