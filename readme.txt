== Description ==

"Job Board Manager - Locations" allows you to integrate location or place feature with jobs, so people can find jobs in their location easily and job archive can be filter by location, like city or country and etc.


### Job Board Manager - Locations by http://pickplugins.com
* [Live Demo &raquo;](http://www.pickplugins.com/demo/job-board-manager/location/dhaka/)
* [Buy Premium &raquo;](http://www.pickplugins.com/item/job-board-manager-locations/)
* [Get Job Board Manager &raquo;](https://wordpress.org/plugins/job-board-manager/)

**Location Page**

This plugin allows you to display single location page, where location details, country, available job list, related locations, Google map and many other information. so people can see available jobs from their location they are interested.

**Location Submission**

User can add their own location to your database, its might now possible to add every location by yourself so our plan to let use contribute for their location want to add. as an admin you have full control to delete or publish the location you want.

`
[job_bm_location_submit_form]
`

**Ajax Location Suggestions**

On job submission page user can get ajax suggestion when start typing, it will help them to find their location by less typing.

**My Locations**

User will see their submitted location via job dashboard, there is shrotcode available to display anywhere.
`
[job_bm_my_locations]
`

###Pro Features

**Job Count by Location**

A widget view for jobs count by location, so visitor will know how many job available right now from their locations.


**Jobs in Map**

Jobs can be display in map view so people will love to see which job available form their map.

**Expandable list by country and city**

A list of jobs by coutry and city can be useful to find all available jobs.


<strong>Translation</strong>

Plugin is translation ready, you can contribute in translation. please contribute here https://translate.wordpress.org/projects/wp-plugins/job-board-manager-locations/

== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Job Board Manager</strong>" activate it.<br />
